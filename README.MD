# World Time Dashboard

A modern, glassmorphism-styled World Clock and Time Management Dashboard built strictly with Vanilla HTML5, CSS3, and JavaScript. No frameworks or external libraries are used.

## Features
- **Local Dashboard:** Live digital and analog clocks with dynamic greetings.
- **World & City Clocks:** Pre-configured with major world countries and cities displaying time, date, GMT offset, and day/night indicators.
- **Favorites System:** Save up to 5 favorite locations (persists via `localStorage`).
- **Real-Time Search:** Instantly filter clocks by country or city name.
- **Time Difference Calculator:** Calculate exactly how many hours/minutes one region is ahead or behind another.
- **Dark/Light Mode:** Toggle between themes seamlessly (preference saved locally).
- **Responsive Design:** Optimized for mobile, tablet, and desktop using CSS Grid & Flexbox.
